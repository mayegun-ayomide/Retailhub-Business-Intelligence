# Business Case 2 — Sales Performance Analysis

## Project Overview

This project analyzes RetailHub sales performance using PostgreSQL.

The objective is to answer business questions related to:
- Revenue performance
- Store performance
- Product sales
- Employee performance
- Customer spending

## Tools Used

- PostgreSQL
- pgAdmin
- SQL

## SQL Concepts Applied

- SELECT
- WHERE
- ORDER BY
- LIMIT
- Aggregate Functions
- GROUP BY
- HAVING
- JOINs

## Business Questions

This case study covers sales analysis questions including:

- Total revenue analysis
- Order performance
- Store performance
- Product performance
- Employee sales analysis
- Customer purchasing behavior

## Dataset

The analysis uses the RetailHub database containing:

- Customers
- Products
- Categories
- Stores
- Employees
- Orders
- Order Items
- Payments
