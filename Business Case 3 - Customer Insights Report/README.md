# Business Case 3 – Customer Insights Report

## Overview

RetailHub's Marketing Department wants to better understand customer behaviour before launching a customer loyalty program.

As the Data Analyst, I was tasked with analyzing customer data to provide insights that support business decision-making.

## Objectives

- Explore customer information
- Analyze customer purchasing behaviour
- Identify high-value customers
- Measure customer spending
- Classify customers using SQL CASE statements

## SQL Concepts Used

- SELECT

- ORDER BY
- LIMIT
- Aggregate Functions
- GROUP BY
- HAVING
- INNER JOIN
- CASE WHEN
