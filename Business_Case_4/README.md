# Business Case 4 – Business Reporting

## Overview

This project contains SQL solutions for Business Case 4 of my SQL portfolio. The case focuses on solving realistic business reporting problems using PostgreSQL while reinforcing SQL JOINs, aggregate functions, GROUP BY, HAVING, CASE expressions, and ORDER BY.

The scenarios simulate requests from business departments such as Finance, Procurement, Inventory, Operations, Customer Experience, HR, Audit, and Executive Management.

---

## Objectives

- Practice SQL JOIN operations
- Build business reporting queries
- Analyze sales and inventory performance
- Perform data validation and auditing
- Develop clean and readable PostgreSQL queries

---

## SQL Concepts

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- GROUP BY
- Aggregate Functions
- HAVING
- CASE WHEN
- ORDER BY
