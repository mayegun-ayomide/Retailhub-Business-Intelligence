# RetailHub Dataset

This project uses a simulated retail database (RetailHub) created for portfolio and learning purposes. It contains tables for customers, products, categories, stores, employees, orders, order items, and payments. The database is designed to simulate realistic retail business operations and support SQL analysis.
This folder contains the SQL script used to create and populate the RetailHub database used throughout this portfolio.

To use this dataset:

1. Open pgAdmin.
2. Create a new database.
3. Open Query Tool.
4. Open `RetailHub_Master.sql`.
5. Execute the script.

The database will be ready for all the business case analyses in this repository.
